hexo图片无法正常显示

修改配置

修改根目录下_config.yml 文件的配置项 post_asset_folder 属性为 true

之后新建博客的时候hexo new "title"的时候会自动生成一个与文章同名的文件夹

需要展示的图片放在这个文件夹中，再进行插入

下载插件

在根目录下在Git Bash中使用npm安装插件

npm install https://github.com/CodeFalling/hexo-asset-image --save

图片插入

前两个步骤完成后,可以进行图片的插入

% asset_img 2.png %外面加一个花括号
